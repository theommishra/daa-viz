# Sketchy Sort - Sorting Algorithm Visualizer

A Next.js application to visualize comparison sorting algorithms with a unique "sketchy" hand-drawn aesthetic.

## Features

- **Algorithms**: Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort.
- **Visuals**: Hand-drawn bars using `roughjs`, smooth reordering animations using `framer-motion`.
- **Controls**: Adjustable array size, speed control, randomization.
- **Stats**: Real-time tracking of comparisons and swaps.

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Rough.js
- Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.
